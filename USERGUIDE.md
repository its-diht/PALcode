
# How to Use

Compile the project with 

Instruction set is described in [the README](README.md).

Write code and save as `<filename.pal>`. 

Run your code via `PAL <filename.pal> [nStacks]`. You will be able to find PAL.exe in the release tab.

If you're trying to run your PAL program with the original PAL java source, run with `java -cp out Main <filename.pal> [nStacks]`. (I'm assuming you all know how to compile a .java file already)
